Andavilla is a high-end luxury villa rental company based in Andalusia, Spain. They were looking for a luxurious, but simple and easy-to-digest website design that would help them become the best choice for people, who are looking to rent a villa in Andalusia.

I came up with this project in order to practice getting more in depth with the interaction and experience design of a website.

Since this was a made-up brand I had to come up with some simple guidelines, so I had something to work with. A quick logotype, colour scheme and typography rules were assembled and everything was set for the web design to begin.

Homepage

The main aims of the landing page are to convert people into Andavilla fans and to make interested users become willing customers. The first part is achieved by utilizing beautiful photography and descriptions of the villas, sharing the company's story and showing testimonials of satisfied customers. And the second part by giving the user a simple and easy way of finding the perfect villa that fits their needs, budget and schedule.

The cover photo focuses on the view from the villa instead of the actual architecture. This would help getting an emotional response from the user (as it makes it easier for them to imagine themselves lounging, relaxing and enjoying the view from the backyard), which should help with conversions. Instead of the cover being a single image from single villa, it could be a slideshow or a video, which would show views from different villas.

To give the user a simple way of finding the perfect villa for their needs, we show them a map with all the villas marked in. Clicking on a marker will show a small card containing information on the villa and a button which starts the booking process. The user can also use filters to make the map only show villas which are available on a certain date, fit their budget and whose description contains certain keywords.

UI

A closer look at some of the elements of the user interface.

Villas

Another way for the user to see all the available villas is the Our Villas page. Here we can see the villas represented as cards, which contain some basic information on each villa. Again we also have a filter, which can help the user narrow down their search to only show the villas which fit their budget or desired date of vacation.

Villa presentation

Each link for an individual villa leads to a special Presentation page of said villa. Here the user can learn more about the villa and its location, see different photos of the house, read reviews left by others and start the booking process.

Checkout

The checkout process was designed to be as easy and simple as possible for the user. The process is split into smaller parts, as to not overwhelm and scare away the user with too many form fields requiring input.

The first part in the process is the selection of the villa and the date of their booking. If they already did that on the specific villa presentation page, this part is already automatically filled in and skipped.

Before proceeding to the next part, if the user is not logged in, a modal window pops up asking them to sign in or register an account.

In the next step the user fills in their personal information. If they already have an account with their details set up, this part is automatically filled in, but not skipped, so the person can make changes before proceeding.

In the third part the user can choose to order extra services, provided by Andavilla or their partners, which are not included in the standard package.

The fourth step asks for the user's payment information. To make it as painless as possible it only requires the user to fill in three fields and it offers detailed information on how to do that.

Lastly the user is asked to review the information they provided, before the order is finalized.

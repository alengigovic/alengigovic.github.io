Lucky's is a made-up brewery, taking some light-hearted inspiration from the prohibition era of the 1920's. The brand is run by nasty gangsters and bootleggers, who (for unknown reasons) wish to stay as disconnected from the company image as possible. So the aim of the project was to create a friendly and approachable visual identity, behind which the owners could quietly operate without raising suspicion of the authorities.

This project was self initiated, with the intent to practice building a visual brand for a company.

Colour & typography

For the main typeface I chose Memphis, mainly because of its warm and friendly character, which is perfect for the brand. It also fits the time-frame of its inspiration as it was designed in 1929 and it does not feel outdated even today.

The secondary typeface is Futura. The sans-serif was chosen because it makes a great partner for Memphis, as they are both geometric typefaces from around the same time. Futura was designed in 1927.

As far as colour goes, things were kept pretty simple. Everything is mostly black and white, with a striking yellow used as the attention grabbing colour.

Bottles

The bottle labels, as most of the brand, were kept pretty simple. This keeps the attention on the beer, instead of it being on the design itself. To differentiate between both of the flavours, the colours were inverted so the difference is easy to spot (even by a person who has had some already).

Website

The primary aims of the website are to provide some information on the company and the product, and directing the users to the store, where they can order the beer and other merchandise. The goals are achieved by implementing a simple design, which uses black and white photography to keep the user interested in viewing the content. It also has distinct calls to action, that guide the user toward the next step in their browsing experience.

The store page features all the items laid out in a grid, to make it easy for the user to quickly sift through the products and buy them. It also has a simple category filter and an easy way to edit the contents of one's shopping cart directly.

The secondary aim of the website is to build an audience of loyal followers. This is done with a blog, where Lucky's employees can share their secrets with the world. The design for the post list is visually quite striking, so it is more likely that the user will check it out. The design for the post focuses more on the actual content. The type is set large, with a comfortable line height and measure so the user can sit back, relax and enjoy reading the post.
